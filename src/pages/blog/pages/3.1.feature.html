FEATURE ENGINEERING

Feature engineering is the process of selecting, transforming, and creating features from raw data to improve the performance of machine learning models.

Useful Techniques

Feature Engineering for Numerical Data

1. Feature Creation - Longlist

Feature creation is the process of creating new features from a dataset to improve the performance of machine learning models.

2. Feature Selection - Shortlist

Feature selection is the process of selecting the most important features from a dataset to improve the performance of machine learning models.

PSI, Information Gain, Missing Value Ratio, Correlation

PSI

PSI - Population Stability Index

PSI tell us the stability of a feature.

Common thresholds are: - PSI < = 0.1, feature is stable - 0.1 > PSI > 0.25, feature is slightly unstable - PSI > 0.25, feature is unstable

Information Gain

Correlation

Correlation is a measure of the relationship between two variables.

Common thresholds are: - Correlation < 0.3, feature is not correlated - 0.3 > Correlation > 0.7, feature is moderately correlated - Correlation > 0.7, feature is highly correlated
